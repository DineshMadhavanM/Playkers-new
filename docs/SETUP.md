# Local Setup

## Prerequisites
- Node.js 20+
- npm 9+

## Install
```
npm install
```

## Environment
Create a `.env` file (see `.env.example`) and set values:
```
PORT=5502
MONGODB_URI=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```

## Run
```
npm start
```
Dev mode (auto-restart):
```
npm run dev
```

## Lint and Format
```
npm run lint
npm run format
```
